# Writeup

This directory contains LaTeX code to reproduce the final draft version of our paper as submitted to Journal of Climate.
It *does not* include changes made since the paper was accepted, including typos and other errors.
If you would like to access the text of our paper, please find final text of our paper available [through open access at the Journal's website](10.1175/JCLI-D-17-0805.1) or [contact me](http://jamesdossgollin.me/#contact).
